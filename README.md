PERTAMA KALI PUSH PROJECT
1. git init == dilakukan 1x di project fresh
2. git add . == memasukan semua file project
3. git commit -m "first commit" == first commit
4. git branch -M main = membuat branch baru
5. git remote add origin <url> ===> git config (autorize akun github)
6. git push -u origin main == push

PULL PROJECT (UPDATE):
1. git add <nama_file>/ git add .
2. git commit
4. git branch -M main
5. git pull origin main

save perubahan di commit:
1. i = INSERT
2. ketikan: initial commit ==> tekan esc
3. ketikan ":wq"

    |

        |
